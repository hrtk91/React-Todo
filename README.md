## ブランチ構成
*develop* -- 開発中のコードをコミットするブランチ
*release* -- 動作確認したコードをデプロイするブランチ

## リポジトリ構成
*Client* -- Reactで作成したクライアント側のコード  
*Database* -- ローカルデバッグ用DBのDockerファイル(※デプロイ対象外)  
*Server* -- ASP.NET Core 6.0 Web APIで作成したApiサーバー
